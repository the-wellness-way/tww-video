VAST/VPAID Plugin by Nuevodevel.com


The basic Vast/Vpaid plugin supports VAST linear video ads. It supports multiple ads delivered through VMAP protocol as well.
Nuevodevel also offers much more advanced, separate VastPro plugin. Apart of linear video ads it supports Vast industry icons, non-linear ads, multiple ad-pod video ads, Vast companions and more.

Code Setup


<!--Load skin css, videojs and plugins inside <body> tag-->
<link href="../skins/shaka/videojs.min.css" rel="stylesheet" type="text/css" />

<script src="../video.min.js"></script>
<script src="../plugins/vast.vpaid.min.js"></script>
<script src="../nuevo.min.js"></script>

<!--Video tag setup-->
<video id="player_1" class="video-js vjs-fluid" controls playsinline poster="../examples/assets/images/poster.jpg">
	<source src="//devnuevo.com/media/video/demo_720.mp4" type="video/mp4">
	<source src="//devnuevo.com/media/video/demo_720.webm" type="video/webm">
</video>

<!--Initialize player and Nuevo pluging-->
<script>
	var player = videojs('player_1');
	player.nuevo();
	<!--Initialize VAST Ad-->
	player.vastAds ({ tagURL: "../examples/assets/vast/spotx.xml", id:"Ad001" });
</script>

Multiple VAST ads

Vast.vpaid plugin supports multiple video linear ads through VMAP protocol. It is akso possible to arrange multiple ads through javascript. 
<script>
player.vastAds ([
   {tagURL: "preroll-1-ad-tag-url", id: 'x01'},
   {tagURL: "preroll-2-ad-tag-url", id: 'x02'},
   {tagURL: "midroll-1-ad-tag-url", id: 'x03', timeOffset:"00:15"},
   {tagURL: "midroll-2-ad-tag-url", id: 'x04', timeOffset:'00:15'},
   {tagURL: "postrol-ad-tag-url", id: 'x04', timeOffset:'end'}
]);
</script>


It is required to setup proper CORS (Cross-Origin Resource Sharing) configuration on server to enable cross-site HTTP requests.
Implementing the request headers to set up CORS correctly depends on the language and framework of the backend.
More information about enabling CORS you can find on https://enable-cors.org
