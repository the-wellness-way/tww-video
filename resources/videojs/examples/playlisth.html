Nuevo Playlist in separate container


It is possible to include Nuevo playlist into any separate container, for example under the player. You only need to provide playlist container's ID and optionally playlist's height (pixels). Once container for the playlist and playlist array are defined you are fine to setup video element, initialize player and Nuevo plugin with the playlist.

Code Source

<!--Setup video element-->
<video id="player_1" class="video-js vjs-fluid" controls playsinline poster="//../poster.jpg"></video>
<div id="playlist"></div>

<!--Initialize player with nuevo plugin and your playlist-->
<script>
	var player = videojs('player_1');
	player.nuevo({ playlistID: "playlist" });
	player.playlist(myplaylist);
</script>

Maximum height of the playlist is 300 pixels. You can easily change it through Nuevo plugin's option playlistMaxH. 
<script>
	var player = videojs('player_1');
	player.nuevo({ playlistID: "playlist", playlistMaxH:400 });
	player.playlist(myplaylist);
</script>

Nuevo playlist features multiple public functions to manage playlist, insert or delete media items programmatically. All functions are described on Nuevodevel playlist SDK demo page.
