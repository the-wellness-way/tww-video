Rewind Button


Nuevo plugin features "Rewind" and "Forward" buttons to rewind or forward video 10 sconds in time.
By default only rewinf button is enabled and displayed. To enable and show forward button you can use simple Nuevo plugin options.
<script>
<!--Initialize player + nuevo and hotkeys plugins-->
	var player1 = videojs('player_1');
	player1.nuevo({ 
		title: "Nuevo plugin for VideoJs Player",
		buttonForward: true
	}); 
</script>

To disable rewind and forward buttons, just disable rewind button. 
<script>
	player.nuevo({ 
		buttonRewind: false
	});
</script>

Rewind and Forward time value is 10 seconds by default. It is possible to set 5, 20, and 30 values instead. 
<script>
	player2.nuevo({ 
		rewindforward:20
	}); 
</script>

Note: Each instance of the player with the nuevo plugin make use of singlePlay enabled option. This allows to play only one player at the same time and pause others automatically.
