VASTPro Plugin by Nuevodevel.com


The VastPro plugin is much more advanced version of the Vast/||Vpaid plugin. Apart of VAST linear video ads it supports VMAP protocol, multiple sequenced Ad Pods, Vast industry icons, Vast companions, non-linear banner ads, non-standard javascript based ads and outstream ads.
More VastPro examples and setup code tutorials you can find on VastPro plugin.

Code Setup


<!--Load skin css, videojs and plugins inside <body> tag-->
<link href="../skins/shaka/videojs.min.css" rel="stylesheet" type="text/css" />

<script src="../video.min.js"></script>
<script src="../nuevo.min.js"></script>
<script src="../plugins/vastpro.js"></script>

<!--Video tag setup-->
<video id="player_1" class="video-js vjs-fluid" controls playsinline poster="../examples/assets/images/poster.jpg">
	<source src="//devnuevo.com/media/video/demo_720.mp4" type="video/mp4">
	<source src="//devnuevo.com/media/video/demo_720.webm" type="video/webm">
</video>

<!--Initialize player and Nuevo pluging-->
<script>
	var player = videojs('player_1');
	player.nuevo();
	<!--Initialize VAST Ad-->
	player.vastAds ({ tagURL: "../examples/assets/vast/vastpro_sample.xml" });
</script>

In video example above you can see 2 preroll ad pod videos, first with industry icon, second with companion banner ad loaded.

Non standard multiple ads

VastPro allows to play linear video ads defined through javascript. It's also possible to define an array of ads and play it at different time position.
