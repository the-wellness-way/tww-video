Nuevo plugin for Videojs framework.

Video.js player file used in `examples` directory is "video.js" or minimized version "video.min.js", located in "videojs" directory.
This is video.js version 8.11.08 released 2024-03-11.
Full version of videojs is located in the same directory.


You may also use any CDN hosted video.js version 7.x or 8.x. Please note that delivered by Nuevodevel.com video.js scripts are slightly modified for better experience when the user drag and move playing progress bar.


Upload "videojs8" directory to your server.
Navigate to "videojs/examples/index.html" for multiple HTML examples of videojs v8 and Nuevo v11, covering most popular videojs and nuevo plugin usage.
